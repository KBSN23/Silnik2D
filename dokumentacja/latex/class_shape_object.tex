\doxysubsection{Detailed Description}
Klasa reprezentujaca obiekt ksztaltu. 

Klasa {\ttfamily \doxylink{class_shape_object}{Shape\+Object}} dziedziczy wirtualnie z {\ttfamily \doxylink{class_drawable_object}{Drawable\+Object}} i {\ttfamily \doxylink{class_transformable_object}{Transformable\+Object}}. Umozliwia polaczenie funkcjonalnosci rysowania i transformacji w obiektach graficznych opartych na ksztaltach, takich jak prostokaty, okregi czy wielokaty.
