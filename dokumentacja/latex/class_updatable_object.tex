\doxysubsection{Detailed Description}
Klasa abstrakcyjna reprezentujaca obiekt, ktory moze byc aktualizowany. 

Klasa ta dziedziczy po klasie \doxylink{class_game_object}{Game\+Object}, umozliwiajac obiektom aktualizacje ich stanu w ramach gry lub aplikacji.\+Klasa jest abstrakcyjna, a konkretne implementacje metody update musza zostac zdefiniowane w klasach pochodnych.
