\doxysubsection{Detailed Description}
Klasa reprezentujaca obiekt bitmapowy. 

Klasa {\ttfamily \doxylink{class_bitmap_object}{Bitmap\+Object}} dziedziczy wirtualnie z {\ttfamily \doxylink{class_drawable_object}{Drawable\+Object}} i {\ttfamily \doxylink{class_transformable_object}{Transformable\+Object}}. Laczy funkcjonalnosci obiektow, ktore mozna rysowac oraz transformowac. Jest przeznaczona do reprezentowania bitmap w aplikacji, umozliwiajac ich rysowanie oraz modyfikacje polozenia, rotacji, czy skalowania
