\doxysubsection{Detailed Description}
Klasa reprezentujaca obiekt sprite\textquotesingle{}u, ktory laczy funkcjonalnosci obrazu bitmapowego oraz animacji. 

Klasa dziedziczy po klasach \doxylink{class_bitmap_object}{Bitmap\+Object} i \doxylink{class_animated_object}{Animated\+Object}, dziêki czemu obiekt sprite\textquotesingle{}u moze zarowno wyswietlac statyczny obraz, jak i obslugiwac animacje.
