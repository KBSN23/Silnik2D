\doxysubsection{Detailed Description}
Klasa bazowa dla obiektow animowanych. 

Klasa {\ttfamily \doxylink{class_animated_object}{Animated\+Object}} definiuje interfejs dla obiektow, ktore maja mozliwosc animacji. Zawiera metode wirtualna czysto abstrakcyjna {\ttfamily animate}, ktora musi zostac zaimplementowana w klasach pochodnych.
