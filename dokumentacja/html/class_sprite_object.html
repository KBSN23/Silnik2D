Detailed Description

Klasa reprezentujaca obiekt sprite'u, ktory laczy funkcjonalnosci obrazu bitmapowego oraz animacji.

Klasa dziedziczy po klasach BitmapObject i AnimatedObject, dziêki czemu obiekt sprite'u moze zarowno wyswietlac statyczny obraz, jak i obslugiwac animacje.
