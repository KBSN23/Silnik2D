Detailed Description

Klasa bazowa dla obiektow animowanych.

Klasa AnimatedObject definiuje interfejs dla obiektow, ktore maja mozliwosc animacji. Zawiera metode wirtualna czysto abstrakcyjna animate, ktora musi zostac zaimplementowana w klasach pochodnych.
