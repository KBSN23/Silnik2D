Detailed Description

Klasa do renderowania prymitywow graficznych.

Klasa PrimitiveRenderer oferuje metody do rysowania pikseli, linii, wielokatow, okrêgow i elips, a takze metody do wypelniania obszarow za pomoca algorytmow takich jak fill.
