Detailed Description

Klasa bazowa dla obiektow, ktore moga byc rysowane.

Klasa DrawableObject definiuje interfejs dla wszystkich obiektow, ktore maja mozliwosc rysowania w oknie SFML.Jest to klasa abstrakcyjna, zawierajaca jedna metode czysto wirtualna draw.

Konstruktor domyslny klasy DrawableObject.

Klasa bazowa reprezentujaca obiekt, ktory mozna narysowac.

Konstruktor inicjalizuje obiekt, ktory moze byc rysowany. Moze byc rozszerzany w klasach pochodnych.

Klasa DrawableObject moze byc podstawa dla innych klas, ktore implementuja funkcje rysowania w graficznym interfejsie uzytkownika.
